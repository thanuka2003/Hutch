# 🧩 Z-Pinger Browser Extension

[![Live Demo](https://img.shields.io/badge/Live-Demo-blue?style=for-the-badge&logo=google-chrome&logoColor=white)](https://kaali-k.github.io/kalanasahan-portfolio/projects_sub/apps_software/extension-z-pinger.html)
[![Extension Status](https://img.shields.io/badge/Status-Beta-orange?style=for-the-badge)](https://github.com/kaali-k/Extension-Z-Pinger)

**Z-Pinger Extension** is a professional network monitoring tool designed as a browser extension. It allows you to monitor network stability, latency (ping), and connectivity status directly from your browser's toolbar without switching windows.

---

## 🚀 Key Features

- **Real-time Latency Tracking**: Instantly see your ping status for any domain.
- **Smart Connection Health**: Visual indicators (Green, Yellow, Red) for network health.
- **Background Execution**: Monitors network status in the background using service workers.
- **Beautiful UI**: Modern, dark-themed popup interface consistent with the Zentix brand.
- **Quick Shortcuts**: Access network tools with a single click.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla CSS3 (Custom Styles), JavaScript (ES6+).
- **Core**: Chrome Extension Manifest V3, Browser Action API, Fetch API for pinging.
- **Icons**: Custom branded icons (128x128, 48x48, 16x16).

## 📦 Installation (Developer Mode)

1. **Download/Clone** this repository.
2. Open **Chrome** and navigate to `chrome://extensions/`.
3. Enable **"Developer mode"** in the top right.
4. Click **"Load unpacked"** and select the folder where you saved this code.
5. Z-Pinger icon will appear in your extensions list!
